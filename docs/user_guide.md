# Healthcare DSS Prototype — React + Vite
This is a **JavaScript + React** implementation of the healthcare DSS: staffing optimization, inventory planning, patient flow analytics, Scenario Explorer, and management ROI.

## Quick start
    Run the follwing commands before running the project
    npm install
    npm install -D tailwindcss@3  
    npm install autoprefixer postcss  
    npm run dev
# open the URL shown (usually http://localhost:5173)
